## 📖 项目简介

**Short-Term Surge Detector** 是一个专为 A 股设计的短线暴涨分析工具，目标捕捉 **10-20 天内 25%+** 的涨幅。

核心特色：
- 🎯 **五维分析引擎**：催化剂/资金/板块/技术/基本面
- ⚠️ **三大失效模式检测**：持仓后每日必检风控
- 🛡️ **严格风控护栏**：硬性止损/时间止损/利润保护

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 基本使用

```python
from SurgeDetector import ShortTermSurgeDetector

detector = ShortTermSurgeDetector()

# 空仓分析
result = detector.analyze("002637")
print(detector.format_report(result))

# 持仓监控
position = {
    'entry_price': 14.5,
    'current_price': 13.2,
    'holding_days': 5
}
result = detector.analyze("002637", position=position)
print(detector.format_report(result))
```

## 📊 分析框架

### 五维分析引擎

| 维度 | 权重 | 检测内容 |
|------|------|----------|
| 催化剂 | 30% | 公告/新闻/政策/业绩预增/并购重组 |
| 资金异动 | 25% | 主力净流入/量价关系/龙虎榜 |
| 板块效应 | 20% | 热点概念匹配/板块排名 |
| 技术形态 | 15% | 均线排列/突破前高/支撑压力 |
| 基本面 | 10% | 业绩/估值/雷区过滤 |

### 三大失效模式

```
┌─────────────────────────────────────────────────────────────┐
│  失效模式 1: 催化剂失效 (Catalyst Invalid)                  │
│  - 公告澄清/否认                                             │
│  - 订单终止或延期超 3 个月                                    │
│  - 业绩预增修正为预减                                        │
│  → CRITICAL: 立即清仓                                        │
├─────────────────────────────────────────────────────────────┤
│  失效模式 2: 资金撤离 (Capital Withdrawal)                  │
│  - 连续 3 日主力资金净流出                                    │
│  - 龙虎榜显示机构大额卖出                                    │
│  - 量价背离 (价涨量缩>30% 或 价跌量增>50%)                    │
│  → HIGH/CRITICAL: 减半持仓/清仓                              │
├─────────────────────────────────────────────────────────────┤
│  失效模式 3: 技术破位 (Technical Breakdown)                 │
│  - 跌破固定止损位 (-8%)                                      │
│  - 跌破关键支撑位 (20 日低点或 MA20 下方 3%)                   │
│  - 时间止损：5 日未启动 (涨幅<5%)                              │
│  - 利润回撤保护：从高点回撤 60%                              │
│  → CRITICAL/MEDIUM: 清仓/减仓                                │
└─────────────────────────────────────────────────────────────┘
```

### 风控护栏

- **硬性止损**: -8% (单票最大亏损)
- **时间止损**: 5 日不启动则离场
- **利润保护**: 从高点回撤 60% 止盈
- **仓位管理**: 根据信号强度自动建议 (NONE/QUARTER/HALF)

## ⚙️ 配置说明

编辑 `config.yaml` 可自定义：
- 触发关键词
- 数据源配置
- 分析权重
- 风控参数
- 失效检测阈值

## 📌 注意事项

1. **数据源**: 依赖 AKShare，需确保网络畅通
2. **实时性**: 数据缓存 5 分钟，收盘前注意刷新
3. **风险提示**: 本工具仅供参考，不构成投资建议
4. **仓位管理**: 单票建议不超过半仓
