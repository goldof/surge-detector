# Short-Term Surge Detector v2.0

> 短线暴涨探测器 - 含三大失效模式检测

## 技能描述

本技能用于分析 A 股股票的短线暴涨潜力，目标捕捉 **10-20 天内 25%+** 的涨幅。  
核心特色是**三大失效模式检测**，确保持仓后每日风控检查。

## 触发方式

```
分析 {股票名称/代码}
{股票代码} 短线怎么样？
{股票名称} 能涨 25% 吗？
```

## 核心功能

### 1. 五维分析引擎

| 维度 | 权重 | 检测内容 |
|------|------|----------|
| 催化剂 | 30% | 公告/新闻/政策/业绩预增/并购重组 |
| 资金异动 | 25% | 主力净流入/量价关系/龙虎榜 |
| 板块效应 | 20% | 热点概念匹配/板块排名 |
| 技术形态 | 15% | 均线排列/突破前高/支撑压力 |
| 基本面 | 10% | 业绩/估值/雷区过滤 |

### 2. 三大失效模式检测（持仓必检）

```
失效模式 1: 催化剂失效 (Catalyst Invalid)
  - 公告澄清/否认
  - 订单终止或延期超 3 个月
  - 业绩预增修正为预减
  - 政策/监管突发利空
  → CRITICAL: 立即清仓

失效模式 2: 资金撤离 (Capital Withdrawal)
  - 连续 3 日主力资金净流出
  - 龙虎榜显示机构大额卖出
  - 量价背离 (价涨量缩>30% 或 价跌量增>50%)
  - 换手率骤降 50% (流动性枯竭)
  → HIGH/CRITICAL: 减半持仓/清仓

失效模式 3: 技术破位 (Technical Breakdown)
  - 跌破固定止损位 (-8%)
  - 跌破关键支撑位 (20 日低点或 MA20 下方 3%)
  - 时间止损：5 日未启动 (涨幅<5%)
  - 利润回撤保护：从高点回撤 60%
  - 墓碑线 (冲高回落>5% 且收阴)
  → CRITICAL/MEDIUM: 清仓/减仓
```

### 3. 风控护栏

- **硬性止损**: -8% (单票最大亏损)
- **时间止损**: 5 日不启动则离场
- **利润保护**: 从高点回撤 60% 止盈
- **仓位管理**: 根据信号强度自动建议 (NONE/QUARTER/HALF)

## 决策规则

```
任一 CRITICAL 失效 → 立即清仓，不问成本
任一 HIGH 失效     → 减半持仓，观察 3 日
评分 < 60 且存在失效 → 回避
评分 > 75 且无失效   → 积极介入
```

## 特殊规则

- **ST 股**: 评分上限 60 分，必须提示风险
- **科创板/创业板**: 涨跌幅 20%，止损调整为 -10%
- **大盘环境**: 上证指数 5 日跌幅>3%，所有评分 -10 分
- **涨停股**: 近 5 日 2 次涨停提示"股性激活"

## 依赖数据源

- **AKShare**: A 股 K 线、实时行情、资金流向、龙虎榜、公司公告
- **备用**: Yahoo Finance (港股/美股)

## 使用示例

```python
from SurgeDetector import ShortTermSurgeDetector

detector = ShortTermSurgeDetector()

# 空仓分析
result = detector.analyze("002637")

# 持仓监控
position = {
    'entry_price': 14.5,
    'current_price': 13.2,
    'holding_days': 5
}
result = detector.analyze("002637", position=position)
```
